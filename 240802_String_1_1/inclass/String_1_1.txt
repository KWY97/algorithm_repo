<오프라인 강의>
split()
 - 문자열 -> 리스트로 변환
 - 파라미터를 기준으로 나누어서 리스트에 넣음

join()
 - 리스트 -> 문자열로 변환
 - iterable(반복 가능한, 순회 가능한) 객체 요소들을 하나의 문자열로 변환
 - 파라미터로 iterable 객체 요소들을 이어줌
 *) join은 속도가 빠른 연산임 (for문 돌려서 순회하면서 append 하는 것 보다 더 빠름)


ladder 문제 풀이 팁
1. 델타 탐색 이용
2. 지나간 길은 0으로 표시
3. 원본이 변경되는 것을 막기 위해 원본을 복사하고 사용
   (import copy, cpoy.deepcopy해도 되지만 cost가 큼, 그래서 [0] 으로 채워진 같은 크기의 리스트인 visited(변수명)를 우선 만듬)
4. 원본에서 방문한 곳은 visited에서 1로 표시
